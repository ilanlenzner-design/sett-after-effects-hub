## üõ†Ô∏è Installation

1.  **Download** this repository.
2.  **Move folder** to your Adobe CEP Extensions directory:
    *   **Mac:** `/Library/Application Support/Adobe/CEP/extensions/`
    *   **Win:** `C:\Program Files (x86)\Common Files\Adobe\CEP\extensions\`
3.  **Enable Scripts:** Open After Effects -> *Settings* -> *Scripting & Expressions* -> Check **"Allow Scripts to Write Files and Access Network"**.
4.  **Restart After Effects**.
5.  Open **Window > Extensions > Smart Text Copywriter**.

## üí° How to Use

1.  **Select a Text Layer** in your composition.
2.  **Type a generic idea** (e.g., "Play now") or use existing text.
3.  **Click a Tool:**
    *   *Analyze & Improve:* Rewrites your text into better ad copy.
    *   *Auto-Highlight:* Colors the important words.
    *   *Smart Wrap:* Fixes the layout.
    *   *Motion Presets:* Adds instant animation.
